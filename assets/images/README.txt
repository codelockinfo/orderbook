PWA ICONS FOLDER
================

This folder should contain the following icon files for PWA functionality:

Required Icon Sizes:
- icon-72.png (72x72 pixels)
- icon-96.png (96x96 pixels)
- icon-128.png (128x128 pixels)
- icon-144.png (144x144 pixels)
- icon-152.png (152x152 pixels)
- icon-192.png (192x192 pixels)
- icon-384.png (384x384 pixels)
- icon-512.png (512x512 pixels)

HOW TO CREATE ICONS:
--------------------

Option 1: Use the Icon Generator
1. Open create-icons.html in your browser
2. Click "Generate Icons" button
3. Right-click each icon and save with the proper name
4. Place all icons in this folder (assets/images/)

Option 2: Use Online Tools
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/

Option 3: Design Your Own
- Use any image editing software (Photoshop, GIMP, etc.)
- Create a square image with your logo
- Export in each required size
- Name files exactly as listed above

IMPORTANT:
----------
The PWA will not work properly without these icon files.
Make sure all icons are PNG format and exactly the sizes specified.
