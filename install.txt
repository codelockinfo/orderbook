ORDER BOOK - INSTALLATION GUIDE
======================================

QUICK START GUIDE
-----------------

1. REQUIREMENTS:
   - WAMP/XAMPP/LAMP server
   - PHP 7.4 or higher
   - MySQL 5.7 or higher
   - Modern web browser

2. INSTALLATION STEPS:

   Step 1: Place Files
   -------------------
   Copy all files to: C:\wamp\www\orderbook\

   Step 2: Create Database
   -----------------------
   a) Open phpMyAdmin (http://localhost/phpmyadmin)
   b) Click "New" to create a database
   c) Database name: orderbook
   d) Collation: utf8mb4_general_ci
   e) Click "Create"

   Step 3: Import Database
   -----------------------
   a) Select the "orderbook" database
   b) Click "Import" tab
   c) Choose file: database.sql
   d) Click "Go"

   Step 4: Configure (if needed)
   -----------------------------
   Open config/database.php and verify:
   - DB_HOST = 'localhost'
   - DB_USER = 'root'
   - DB_PASS = '' (empty for WAMP)
   - DB_NAME = 'orderbook'

   Open config/config.php and verify:
   - BASE_URL = 'http://localhost/orderbook/'

4. TROUBLESHOOTING:

   Problem: Blank page
   Solution: Check PHP error logs in WAMP

   Problem: Database connection error
   Solution: Verify database credentials in config/database.php

   Problem: Can't login
   Solution: Make sure database was imported correctly

   Problem: PWA not working
   Solution: Create icon files in assets/images/ folder
